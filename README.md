# 论坛项目 - 第一阶段

## 项目概述
建设一个论坛系统，第一阶段包含用户管理和新闻发布功能。

## 技术栈
- **前端**: Vue3 + TypeScript + Vite + Element Plus
- **后端**: Java + Spring Boot 3
- **数据库**: PostgreSQL 15+
- **权限**: Spring Security + JWT + 按钮级权限控制
- **登录**: 账号密码 + 微信OAuth
- **部署**: 本地服务器 + Docker

## 功能模块

### 用户管理
- 注册/登录（账号密码 + 微信）
- 角色管理（管理员、普通用户）
- 权限管理（菜单级 + 按钮级）
- 用户信息维护

### 新闻发布
- 新闻CRUD（富文本编辑器）
- 分类/标签管理
- 新闻列表展示
- 新闻详情页

### 系统功能
- 操作日志
- 系统配置
- 文件上传
