## 技术栈覆盖

### 后端技术
- **框架**: Express.js
- **数据库**: PostgreSQL + TypeORM
- **认证**: JWT + bcryptjs
- **验证**: Zod
- **缓存**: Redis
- **邮件**: Nodemailer

### 前端技术
- **HTTP客户端**: Axios
- **状态管理**: localStorage/sessionStorage
- **类型安全**: TypeScript
