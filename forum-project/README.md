# 论坛开发项目

## 项目概述
开发一个功能完整的在线论坛系统

## 项目目标
- 用户注册、登录、权限管理
- 帖子发布、回复、编辑、删除
- 分类管理、标签系统
- 搜索功能
- 用户个人中心
- 管理员后台

## 技术栈
- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express + TypeScript
- 数据库：PostgreSQL
- 认证：JWT
- 部署：Docker + Nginx

## 项目结构
```
forum-project/
├── frontend/          # 前端代码
├── backend/           # 后端代码
├── docker/           # Docker配置
├── docs/             # 项目文档
└── scripts/          # 部署脚本
```
